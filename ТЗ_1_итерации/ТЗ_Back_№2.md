### **0. ИСПОЛЬЗУЕМАЯ БИБЛИОТЕКА ДЛЯ ВИЗУАЛЬНОГО ГРАФА**

Для отображения визуального графа используется **NodeGraphProcessor** — библиотека с открытым исходным кодом.

| Параметр | Значение |
|----------|----------|
| **Название** | NodeGraphProcessor |
| **Автор** | alelievr (оригинал), форк Warwlock |
| **Используемый форк** | **Warwlock/NodeGraphProcessor** (совместим с Unity 6) |
| **GitHub репозиторий** | https://github.com/Warwlock/NodeGraphProcessor |
| **Установка** | Package Manager → Add package from git URL: `https://github.com/Warwlock/NodeGraphProcessor.git` |
| **Лицензия** | MIT |
| **Версия** | последняя (совместима с Unity 2022.3+) |

**Почему выбран этот форк:**
- Оригинальная библиотека alelievr не обновлялась с 2021 года
- Форк Warwlock поддерживает актуальные версии Unity (2022.3+)
- Исправлены проблемы с `EdgeDragHelper` и другими API, устаревшими в Unity 6

**Интеграция:**
- Плагин использует `BaseGraph` и `BaseGraphView` из библиотеки
- Узлы наследуются от `GraphProcessor.BaseNode`
- Атрибуты `[Input]`, `[Output]`, `[NodeMenuItem]` используются для создания портов и меню

### **1. ИСХОДНОЕ СОСТОЯНИЕ (что было до начала работ)**

#### **1.1. Архитектура**
- Плагин представлял собой набор скриптов в папке `Assets/Plugins/CustomVisualScripting/`
- Отдельный C# проект `VisualScripting.Core/` использовался для компиляции DLL
- Отсутствовала единая структура для распространения

#### **1.2. Функциональность**
| Компонент | Что работало |
|-----------|--------------|
| Парсер | Только простейшая арифметика `x = y + z` через Regex |
| Генератор | Генерировал код с временными именами `temp0`, `temp1` |
| Визуальный граф | Отсутствовал, вместо него текстовый список нод |
| Сохранение/загрузка | Работало через JSON |
| Работа с файлами | Только кнопки "Сохранить" и "Загрузить" |
| Ноды | 18 типов (литералы, арифметика, сравнения, if, debug, unity, переменные) |

#### **1.3. Проблемы**
1. **Парсер на Regex** — не поддерживал приоритеты операций, скобки, многострочные выражения
2. **Отсутствие связей** — `GraphData.Edges` всегда пуст
3. **Нет имен переменных** — при генерации терялись имена `x`, `y`, `z`
4. **Нет визуального графа** — пользователь не мог перетаскивать ноды и соединять порты
5. **Нет полноценной работы с файлами** — отсутствовали "Новый", "Сохранить как"
6. **Ограниченная поддержка конструкций** — только `+`, `-`, `*`, `/`, простой `if`

### **3. ЦЕЛЕВОЕ СОСТОЯНИЕ (MVP)**

#### **3.1. Арифметика**

| Оператор | Приоритет | Пример | Генерация |
|----------|-----------|--------|-----------|
| `*`, `/`, `%` | 1 (высокий) | `x * y` | `x * y` |
| `+`, `-` | 2 (низкий) | `x + y` | `x + y` |
| `( )` | всегда | `(x + y) * z` | `(x + y) * z` |

#### **3.2. Сравнения и логика**

| Оператор | Тип | Генерация |
|----------|-----|-----------|
| `>=` | CompareGreaterOrEqual | `x >= y` |
| `<=` | CompareLessOrEqual | `x <= y` |
| `!=` | CompareNotEqual | `x != y` |
| `&&` | LogicalAnd | `x && y` |
| `||` | LogicalOr | `x || y` |
| `!` | LogicalNot | `!x` |

#### **3.3. Условные конструкции**

```csharp
if (условие)
{
    // тело
}
else if (условие)
{
    // тело
}
else
{
    // тело
}
```

#### **3.4. Визуальный граф**

- Отображение нод с портами ввода/вывода
- Перетаскивание нод мышкой
- Соединение портов для создания связей
- Редактирование значений нод в инспекторе

#### **3.5. Работа с файлами**

| Кнопка | Действие |
|--------|----------|
| **Новый** | Очистить всё, запросить имя, сохранить пустой граф |
| **Открыть** | Выбрать JSON, загрузить граф и код в редактор |
| **Сохранить** | Сохранить в текущий файл |
| **Сохранить как** | Запросить новое имя, сохранить |
| **Парсить код** | Преобразовать код в ноды (код не исчезает) |
| **Сгенерировать** | Преобразовать ноды в код (ноды не исчезают) |

### **8. РИСКИ И МИТИГАЦИИ**

| Риск | Вероятность | Митигация |
|------|-------------|-----------|
| Roslyn сложнее, чем ожидалось | Средняя | Начать с простых выражений, постепенно усложнять |
| Проблемы с приоритетами операторов | Средняя | Использовать алгоритм Shunting Yard |
| Конфликты имен с NodeGraphProcessor | Низкая | Использовать полные имена или алиасы |
| Несовместимость версий Unity | Низкая | Тестировать на целевой версии 6000.4.0f1 |
| Newtonsoft.Json не установлен | Низкая | Добавить в инструкцию по установке |
