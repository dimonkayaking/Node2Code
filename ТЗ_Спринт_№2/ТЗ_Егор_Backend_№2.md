## 4. Технические требования

### 4.1. Формат нод для циклов

#### ForNode
```csharp
public class ForNode : CustomBaseNode
{
    [Input("execIn")] public Flow execIn;
    [Input("init")] public object init;
    [Input("condition")] public bool condition;
    [Input("increment")] public object increment;
    [Output("body")] public Flow body;
    [Output("execOut")] public Flow execOut;
}
```

#### WhileNode
```csharp
public class WhileNode : CustomBaseNode
{
    [Input("execIn")] public Flow execIn;
    [Input("condition")] public bool condition;
    [Output("body")] public Flow body;
    [Output("execOut")] public Flow execOut;
}
```

### 4.2. Формат нод для встроенных методов

#### ConsoleWriteLineNode
```csharp
public class ConsoleWriteLineNode : CustomBaseNode
{
    [Input("execIn")] public Flow execIn;
    [Input("message")] public string message;
    [Output("execOut")] public Flow execOut;
}
```

#### ParseNode (int / float)
```csharp
public class IntParseNode : CustomBaseNode
{
    [Input("execIn")] public Flow execIn;
    [Input("input")] public string input;
    [Output("result")] public int result;
    [Output("execOut")] public Flow execOut;
}
```

### 4.3. Пример генерации цикла `for`

**Входной граф:**
- `ForNode` с `init = "int i = 0"`, `condition = "i < 10"`, `increment = "i++"`, `body` содержит `AddNode`

**Ожидаемый код:**
```csharp
for (int i = 0; i < 10; i++)
{
    sum = sum + i;
}
```

### 4.4. Пример генерации `Console.WriteLine`

**Входной граф:**
- `ConsoleWriteLineNode` с `message = "Hello"`

**Ожидаемый код:**
```csharp
Console.WriteLine("Hello");
// или
Debug.Log("Hello");
```
