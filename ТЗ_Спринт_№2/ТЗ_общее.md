# Техническое задание (итоговое)
## Проект: Плагин визуального скриптинга для Unity + обучающая платформа

**Версия:** 3.0 (финальная перед предзащитой)  
**Дата:** апрель 2026  
**Статус:** подготовка к предзащите (16 апреля)

---

## 1. Цель проекта

Разработать экосистему для обучения программированию и визуальной разработки в Unity, включающую:

1. **Плагин для Unity** — двусторонний конвертер между C# кодом и визуальным графом (ноды)
2. **Обучающий курс** — структурированная программа от основ до продвинутых тем
3. **Сайт-платформа** — с адаптивным дизайном, личным кабинетом, базой данных пользователей и прогресса

---

## 2. Что уже сделано

### 2.1. Плагин для Unity

#### Backend 1 (Салават) — парсер
- Парсер C# → граф на Roslyn (500+ строк)
- Модели данных: `GraphData`, `NodeData`, `NodeType`, `ParseResult`
- 7 новых типов нод: `%`, `>=`, `<=`, `!=`, `&&`, `||`, `!`
- Создание связей (Edges) между нодами
- Сохранение имён переменных (`VariableName`)

#### Backend 2 (Егор) — генератор + ноды
- 23 базовые ноды (литералы, математика, сравнения, if, Unity, переменные, debug)
- Генератор кода из графа (400+ строк) с топологической сортировкой
- Работа с файлами: «Новый», «Сохранить», «Сохранить как», «Загрузить»
- Сохранение и восстановление позиций нод

#### Fullstack (Дима) — визуальный граф + UI
- Подключение библиотеки NodeGraphProcessor (форк Warwlock)
- Главное окно плагина (600+ строк)
- Отображение нод, перетаскивание мышкой
- Синхронизация значений и названий нод ✅ (исправлено)
- Обновление всех 30+ узлов
- UI компоненты: редактор кода, панель ошибок

### 2.2. Сайт и дизайн (Денис)
- 8 страниц в Figma (главная, вход/регистрация, плагин, курс, уроки, профиль, подтверждение выхода)
- Адаптив: десктоп (1440px), планшет (768px), телефон (375px)
- Светлая и тёмная темы
- Компоненты: карточки тем, строки уроков, кнопки (3 типа), поля ввода
- **Полноценная реализация сайта на React** ✅ (готова)

### 2.3. Курс и тестирование (Кирилл)
- Программа курса: 8 модулей, 24 урока
- Тестирование плагина, отчёт по багам
