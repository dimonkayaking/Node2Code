# 📋 ИТОГОВОЕ ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ТЗ)
## Visual Scripting Plugin для Unity на базе Node Graph Processor

---

## 📌 ОБЩАЯ ИНФОРМАЦИЯ

**Название проекта:** Visual Scripting Plugin  
**Цель:** Создать двусторонний мост между C# кодом и визуальными нодами в Unity  
**Срок:** 2 месяца до MVP  
**Команда:** 3 разработчика  
**Целевая версия Unity:** 2022.3 LTS и выше  
**Технологии:** Unity, Node Graph Processor, .NET 10, Roslyn, Newtonsoft.JSON

#### 📦 Категория: Литералы (4 шт) - простые

#### 📦 Категория: Сравнение (3 шт) - средние
- `GreaterNode.cs` - больше (>)
- `LessNode.cs` - меньше (<)

#### 📦 Категория: Переменные (3 шт) - сложные
- `VariableDeclarationNode.cs` - объявление переменной
- `GetVariableNode.cs` - чтение переменной
- `SetVariableNode.cs` - присваивание

#### 📦 Категория: Управление (1 шт) - очень сложная
- `IfNode.cs` - условие (с Execution портами)

#### 📦 Категория: Unity (3 шт) - средние
- `Vector3CreateNode.cs` - new Vector3(x, y, z)
- `GetPositionNode.cs` - transform.position (чтение)
- `SetPositionNode.cs` - transform.position = (запись)

#### 📦 Категория: Вывод (1 шт) - средняя
