## **План разработки плагина Custom Visual Scripting**
**Документ для команды**

---

### **1. ИСХОДНОЕ СОСТОЯНИЕ (что было до начала работ)**

#### **1.1. Архитектура**
- Плагин представлял собой набор скриптов в папке `Assets/Plugins/CustomVisualScripting/`
- Отдельный C# проект `VisualScripting.Core/` использовался для компиляции DLL
- Отсутствовала единая структура для распространения

#### **1.2. Функциональность**
| Компонент | Что работало |
|-----------|--------------|
| Парсер | Только простейшая арифметика `x = y + z` через Regex |
| Генератор | Генерировал код с временными именами `temp0`, `temp1` |
| Визуальный граф | Отсутствовал, вместо него текстовый список нод |
| Сохранение/загрузка | Работало через JSON |
| Работа с файлами | Только кнопки "Сохранить" и "Загрузить" |
| Ноды | 18 типов (литералы, арифметика, сравнения, if, debug, unity, переменные) |

#### **1.3. Проблемы**
1. **Парсер на Regex** — не поддерживал приоритеты операций, скобки, многострочные выражения
2. **Отсутствие связей** — `GraphData.Edges` всегда пуст
3. **Нет имен переменных** — при генерации терялись имена `x`, `y`, `z`
4. **Нет визуального графа** — пользователь не мог перетаскивать ноды и соединять порты
5. **Нет полноценной работы с файлами** — отсутствовали "Новый", "Сохранить как"
6. **Ограниченная поддержка конструкций** — только `+`, `-`, `*`, `/`, простой `if`

### **3. ЦЕЛЕВОЕ СОСТОЯНИЕ (что должно быть)**

#### **3.1. Арифметика (полная поддержка)**

| Оператор | Приоритет | Пример | Генерация |
|----------|-----------|--------|-----------|
| `**` | 1 (высокий) | `x ** y` | `Math.Pow(x, y)` |
| `*`, `/`, `%` | 2 (средний) | `x * y` | `x * y` |
| `+`, `-` | 3 (низкий) | `x + y` | `x + y` |
| `( )` | всегда | `(x + y) * z` | `(x + y) * z` |

#### **3.2. Сравнения и логика**

| Оператор | Тип | Генерация |
|----------|-----|-----------|
| `>=` | CompareGreaterOrEqual | `x >= y` |
| `<=` | CompareLessOrEqual | `x <= y` |
| `!=` | CompareNotEqual | `x != y` |
| `&&` | LogicalAnd | `x && y` |
| `||` | LogicalOr | `x || y` |
| `!` | LogicalNot | `!x` |

#### **3.3. Циклы**

| Конструкция | Нода | Генерация |
|-------------|------|-----------|
| `for (int i = 0; i < n; i++)` | ForNode | `for (int i = 0; i < n; i++) { ... }` |
| `while (условие)` | WhileNode | `while (условие) { ... }` |

#### **3.4. Визуальный граф**

- Отображение нод с портами ввода/вывода
- Перетаскивание нод мышкой
- Соединение портов для создания связей
- Редактирование значений нод в инспекторе

#### **3.5. Работа с файлами**

| Кнопка | Действие |
|--------|----------|
| **Новый** | Очистить всё, запросить имя, сохранить пустой граф |
| **Открыть** | Выбрать JSON, загрузить граф и код в редактор |
| **Сохранить** | Сохранить в текущий файл |
| **Сохранить как** | Запросить новое имя, сохранить |
| **Парсить код** | Преобразовать код в ноды (код не исчезает) |
| **Сгенерировать** | Преобразовать ноды в код |
