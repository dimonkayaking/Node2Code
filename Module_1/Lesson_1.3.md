# Теория к уроку 1.3: Знакомство с плагином

## Цель урока
Научиться импортировать плагин визуального скриптинга в проект Unity, открывать окно редактора нод и ориентироваться в его интерфейсе.

---

## 1. Что такое плагин и как его добавить
Плагин — это дополнительный модуль, который расширяет базовый функционал Unity. В нашем случае он добавляет возможность создавать скрипты с помощью визуальных блоков (нод), не написав ни строчки кода вручную.

### 1.1. Импорт плагина
1. Убедитесь, что ваш проект в Unity открыт.
2. Найдите скачанный файл плагина на вашем компьютере (обычно такие ассеты имеют расширение `.unitypackage`).
3. Дважды кликните по файлу **ИЛИ** перетащите его прямо в окно **Project** в Unity.
4. Откроется окно **Import Unity Package**. Убедитесь, что галочками отмечены все файлы в списке.
5. Нажмите кнопку **Import** в правом нижнем углу.
6. Дождитесь окончания загрузки. В окне Project появится новая папка с файлами плагина.

---

## 2. Открытие окна визуального редактора
Как и любое другое окно в Unity (Scene, Console), интерфейс нашего плагина открывается через верхнее меню.

1. На верхней панели Unity нажмите **Tools**.
2. Найдите в выпадающем списке *Visual Scripting*.
3. Появившееся окно можно прикрепить рядом с окном Scene или Game для удобства работы, просто потянув за его название (вкладку).

---

## 3. Основные элементы интерфейса плагина

Интерфейс редактора нод состоит из нескольких ключевых зон.

| Элемент | Назначение |
|---------|------------|
| **Верхняя панель** | Содержит кнопки для генерации C# кода, генерации нодового дереева, сохранения и загрузки программы, очитски холста. |
| **Холст (Canvas)** | Главная рабочая область. Бесконечное пространство, где вы размещаете ноды и соединяете их «проводами» (связями). |
| **Панель поиска (Search Menu)** | Меню для добавления новых блоков. Вызывается кликом правой кнопки мыши по пустому месту на холсте и нажатииеем Create Node. Содержит категории нод и строку поиска. |
| **Порты (Ports)** | Точки подключения по бокам ноды. Левые порты принимают данные и сигналы (Inputs), правые — передают их дальше (Outputs). |
| **C# Code** | Код сгенированный из нод (или тот из которого необходимо сгенироровать нодовое дерево). |

---

## 4. Базовые действия с нодами

- **Навигация по холсту:** Нажмите среднюю кнопку мыши (колесико) и потяните, чтобы перемещаться по холсту. Покрутите колесико для масштабирования (Zoom).
- **Добавление ноды:** Кликните правой кнопкой мыши по холсту, нажмите Create Node, начните вводить название нужной ноды в поиск и выберите её из появившегося списка.
- **Выделение и перемещение:** Кликните левой кнопкой мыши по заголовку ноды и потяните, чтобы переместить её в удобное место.
- **Создание связи:** Нажмите левой кнопкой мыши на выходном порту одной ноды и протяните линию к входному порту другой ноды.
- **Удаление:** Выделите ноду или связь (кликнув по ней) и нажмите клавишу `Delete` на клавиатуре.

---

## 5. Шпаргалка (краткая памятка)

- **Импорт:** Перетащить `.unitypackage` в окно Project → нажать Import.
- **Запуск:** Открыть редактор через верхнее меню `Tools`.
- **Холст:** Бесконечная доска для создания логики. Перемещение — средняя кнопка мыши.
- **Ноды:** Визуальные блоки логики. Добавляются через правый клик по холсту.
- **Связи:** Линии между портами нод, передающие информацию или команду к действию от одного блока к другому.
