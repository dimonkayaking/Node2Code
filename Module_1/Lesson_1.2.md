# Теория к уроку 1.2: Обзор интерфейса Unity

## Цель урока
Ознакомиться с основными окнами редактора Unity, научиться ориентироваться в интерфейсе и понимать назначение каждого элемента.

---

## 1. Основные окна редактора

Unity использует систему настраиваемых окон. Ниже перечислены окна, которые необходимы на начальном этапе.

| Окно | Назначение |
|------|------------|
| **Scene** | Редактирование сцены: размещение, перемещение и изменение объектов. |
| **Game** | Предварительный просмотр игры с точки зрения камеры. Видно то, что увидят игроки. |
| **Hierarchy** | Список всех объектов на текущей сцене. Показывает вложенность (родитель-дочерние связи). |
| **Inspector** | Параметры выбранного объекта: компоненты, свойства, настройки. (Подробная работа с Inspector будет в следующих уроках.) |
| **Project** | Файловая система проекта: скрипты, сцены, текстуры, префабы. |
| **Console** | Вывод сообщений, предупреждений и ошибок. |

---

## 2. Навигация и работа с окнами

- Окна можно перемещать, перетаскивая за заголовок.
- При перетаскивании появляются подсказки для закрепления окна (dock hints).
- Любое окно можно открыть через меню `Window`. Например: `Window → General → Console` открывает окно Console, `Window → General → Project` — окно Project.

---

## 3. Панель инструментов (Toolbar)

В верхней части редактора расположены основные элементы управления:

| Элемент | Действие |
|---------|----------|
| **Hand Tool** | Панорамирование сцены (перемещение вида). |
| **Move Tool** | Перемещение выделенного объекта. |
| **Rotate Tool** | Вращение выделенного объекта. |
| **Scale Tool** | Масштабирование выделенного объекта. |
| **Rect Tool** | Трансформирование 2D-объектов (изменение размеров прямоугольной области). |
| **Play (кнопка ▶)** | Запуск игры в редакторе. Повторное нажатие останавливает. |
| **Pause (⏸)** | Приостановка игры. |

---

## 4. Ключевые действия в окне Scene

- **Выделить объект** – кликнуть на объекте в Scene или Hierarchy.
- **Переместить объект** – выбрать Move Tool, потянуть за оси (красный = X, зелёный = Y, синий = Z).
- **Вращать объект** – выбрать Rotate Tool, потянуть за дуги.
- **Масштабировать** – выбрать Scale Tool, потянуть за кубики на осях или центральный куб для равномерного масштаба.
- **Сфокусироваться на объекте** – выделить объект в Hierarchy и нажать кнопку с изображением лупы (Frame Selected) в правом верхнем углу окна Scene.

## 6. Управление проектом (окно Project)

В окне Project отображается содержимое папки `Assets` на диске.
- **Создание папок/ресурсов:** правой кнопкой мыши в окне Project → `Create`.
- **Импорт файлов:** перетащить файлы из проводника в окно Project.
- **Поиск:** строка поиска фильтрует по имени.

---

## 7. Запуск и остановка игры

1. Нажать кнопку **Play** (▶) на панели инструментов.
2. Редактор переходит в режим игры. Окно Game показывает вид камеры.
3. В режиме игры можно вносить изменения, но они **не сохраняются** после остановки.
4. Нажать **Play** повторно, чтобы выйти из режима игры.

---

## 8. Шпаргалка (краткая памятка)

- **Scene** – редактирование мира.
- **Game** – вид от камеры.
- **Hierarchy** – список всех объектов на сцене.
- **Inspector** – настройки выделенного объекта (Transform и др.).
- **Project** – файлы проекта (Assets).
- **Console** – сообщения и ошибки.
- **Move / Rotate / Scale** – инструменты трансформации.
- **Play (▶)** – запуск игры.
