# Теория к уроку 1.1: Установка Unity

## Цель урока
Установить Unity Hub, установить редактор Unity 6, создать новый проект с шаблоном **2D URP**.

---

## 1. Компоненты установки

### 1.1. Unity Hub
Unity Hub — приложение для управления установленными версиями Unity, проектами и лицензиями.

### 1.2. Редактор Unity 6
Основная программа, в которой ведётся разработка. Для курса используется **Unity 6**.

### 1.3. Модули
При установке редактора можно добавить дополнительные компоненты.  
**Обязательный минимум:** модуль сборки для вашей операционной системы (Windows / macOS / Linux). Остальные модули можно добавить позже.

---

## 2. Пошаговая установка

### 2.1. Скачать и установить Unity Hub
1. Перейти на [unity.com/download](https://unity.com/download).
2. Нажать **“Download Unity Hub”**.
3. Выбрать установщик для своей ОС, запустить его и следовать инструкциям.

### 2.2. Установить редактор Unity 6 через Hub
1. Открыть Unity Hub.
2. Перейти на вкладку **Installs** (Установки).
3. Нажать кнопку **“Install Editor”**.
4. В списке версий выбрать **Unity 6** (например, Unity 6000.0.x).
5. В окне выбора модулей отметить:
   - **Обязательно:** модуль для вашей платформы (Windows Build Support / Mac Build Support / Linux Build Support).
   - Все остальные модули можно не выбирать.
6. Нажать **“Continue”** и дождаться окончания загрузки и установки.

### 2.3. Активация бесплатной лицензии
1. В Unity Hub нажать на иконку шестерёнки (Settings) → **License Management**.
2. Нажать **“Activate New License”**.
3. Выбрать **“Unity Personal”** → **“Activate”**.

Лицензия активируется автоматически после входа в учётную запись Unity.

---

## 3. Создание первого проекта

1. В Unity Hub перейти на вкладку **Projects**.
2. Нажать **“New Project”**.
3. В списке шаблонов выбрать **“2D URP”** (2D с Universal Render Pipeline).
4. Заполнить поля:
   - **Project name** — имя проекта (латиницей, без специальных символов).
   - **Location** — папка, где будет сохранён проект.
5. В поле **“Editor Version”** убедиться, что выбрана установленная Unity 6.
6. Нажать **“Create project”**.

После создания проекта откроется редактор Unity с готовой сценой, настроенной под 2D-рендеринг URP.

---

## 4. Проверка после установки

| Что проверить | Как проверить |
|---------------|---------------|
| Проект открывается без ошибок | В редакторе видна сцена, окна (Scene, Game, Hierarchy, Project, Inspector) отображаются |
| В консоли нет красных ошибок | Открыть окно Console (Window → General → Console) |
| На сцене есть камера | В Hierarchy присутствует объект Main Camera |

---

## 5. Шпаргалка (краткая памятка)

1. **Установить Unity Hub** с сайта unity.com.  
2. **Установить Unity 6** через Hub: Installs → Install Editor → выбрать Unity 6 → добавить модуль сборки для вашей ОС.  
3. **Активировать** лицензию Unity Personal.  
4. **Создать проект:** Projects → New Project → выбрать шаблон **2D URP** → указать имя и место → создать.  
5. Убедиться, что редактор открылся и нет ошибок.
