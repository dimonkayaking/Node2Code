## 2. ЦЕЛИ И ЗАДАЧИ ПРОЕКТА

### 2.1. Основная цель

Разработать модуль для игрового движка Unity, который позволяет создавать игровую логику двумя способами: через визуальные ноды и через написание кода на C#, с возможностью двусторонней конвертации между этими форматами.

### 2.2. Задачи проекта

1. Разработать парсер C# кода в граф нод на базе Roslyn
2. Разработать генератор C# кода из графа нод
3. Создать редактор визуальных нод на основе GraphProcessor
4. Реализовать более 30 типов нод
5. Разработать систему выполнения графов в реальном времени
6. Создать обучающий сайт с курсом из 8 модулей и 24 уроков, с полной реализацией первого модуля
7. Разработать бэкенд для хранения пользователей и прогресса
8. Подготовить документацию и презентационные материалы

### 2.3. Целевая аудитория

- Школьники, студенты, самоучки
- Преподаватели информатики

## 5. РЕЗУЛЬТАТЫ ПРОЕКТА

### 5.1. Unity плагин Node2Code

Разработан плагин для Unity, который включает:

- Редактор визуальных нод на базе GraphProcessor
- Парсер C# кода в граф на базе Roslyn
- Генератор C# кода из графа
- Более 30 типов нод:
  - Литералы (int, float, bool, string)
  - Математика (+, -, *, /, %, Mathf.Abs, Mathf.Max, Mathf.Min)
  - Сравнения (==, !=, >, <, >=, <=)
  - Логика (&&, ||, !)
  - Управление потоком (if, else, for, while, Console.WriteLine)
  - Конвертация (int.Parse, float.Parse, ToString)
- Экспорт и импорт графов в .cs

### 5.2. Обучающий сайт

Разработан веб-сайт, который включает:

- Лендинг с описанием продукта
- Страница плагина со скачиванием
- Курс с готовым 1 модулем с отслеживанием прогресса обучения
- Личный кабинет пользователя с регистрацией и авторизацией
- Тёмная и светлая тема оформления

### 5.3. Бэкенд

Разработана серверная часть, которая включает:

- API для регистрации и входа пользователей
- API для хранения и получения прогресса
- База данных PostgreSQL
- Хеширование паролей (scrypt)


## 6. ТЕХНИЧЕСКАЯ АРХИТЕКТУРА

### 6.1. Общая архитектура

Проект состоит из трёх основных компонентов:

1. Unity плагин (ядро продукта)

### 6.4. Технологии бэкенда сайта

- Node.js + Express
- PostgreSQL
- Scrypt (хеширование паролей)


## 7. РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

### 7.1. Установка плагина

1. Скачайте файл Node2Code.unitypackage с сайта
2. Откройте ваш проект в Unity
3. Нажмите **Assets → Import Package → Custom Package**
4. Выберите скачанный файл
5. Нажмите **Import**
6. Откройте **Window → Package Manager**
7. Нажмите кнопку **+** (плюс) в левом верхнем углу
8. Выберите **Add package from git URL...**
9. Вставьте ссылку: `https://github.com/Warwlock/NodeGraphProcessor.git`
10. Нажмите **Add**
11. В Package Manager снова нажмите **+**
12. Выберите **Add package by name...**
13. Введите: `com.unity.nuget.newtonsoft-json`
14. Нажмите **Add**
15. Откройте плагин: в меню Unity выберите **Tools → Node2Code**

### 7.2. Создание первого графа

1. Откройте **Tools → Node2Code**
2. Нажмите **"Создать новый граф"**
3. Через ПКМ → **Create Node → Тип ноды → Нода** создайте ноды
4. Соедините порты нод, перетаскивая мышью от одного порта к другому
5. Нажмите **"Сгенерировать код"** — получите C# скрипт
6. Нажмите **"Run"** — граф запустится прямо в окне редактора
