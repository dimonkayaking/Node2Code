# Node2Code

**Краткое описание продукта**

Node2Code — это плагин для Unity, который позволяет создавать игровую логику в двух форматах: визуальные ноды и код C#. Главная особенность — двусторонняя конвертация между этими форматами.

**Что умеет продукт:**

- Рисовать логику в виде нод и получать готовый C# код
- Парсить написанный C# код обратно в визуальный граф
- Выполнять графы прямо в редакторе Unity
- Сохранять и загружать графы в JSON

**Комплект поставки:**

- Плагин для Unity (30+ типов нод)
- Обучающий сайт с курсом из 8 модулей и 24 уроков (реализован полностью 1 модуль)
- Личный кабинет с отслеживанием прогресса

**Для кого:**

- Начинающие разработчики (школьники, студенты, самоучки)
- Геймдизайнеры для быстрого прототипирования
- Преподаватели информатики
