О проекте «Самописный модуль для игрового движка»
Краткое название:
Самописный модуль для игрового движка
Название для диплома:
Разработка модуля для игрового движка
Уровень сложности:
A
Тип проводимых работ:
Прикладной (практико-ориентированный)
Описание:
Команда разрабатывает модуль (например: система освещения, ИИ, физика, сетевой режим). Проект позволяет углубить знания в архитектуре игровых движков и программировании.

Цель:
Разработать собственный программный модуль для расширения функциональности игрового движка.

Требуемый результат:
Программный модуль (плагин, библиотека) с документацией и примерами использования.

Критерии оценки:
Модуль корректно подключается к движку

Реализована заявленная функциональность

Отсутствуют критические ошибки

Есть документация и инструкция

Присутствуют демонстрационные сцены/проекты
